Core Features

개발자 없이도,
이 모든 것이 가능합니다

SpeedMIS는 복잡한 업무 시스템을 누구나 쉽게 만들 수 있도록 설계되었습니다.

📋

노코드 프로그램 생성

메뉴와 필드를 정의하면 목록, 조회, 수정, 삭제 화면이 자동 생성됩니다. SQL을 몰라도 됩니다.

🔐

정교한 권한 관리

그룹별, 사용자별, 프로그램별 세밀한 접근 권한. 부서/직급 기반 자동 필터링까지.

⚡

실시간 목록편집

그리드에서 직접 값을 수정하고 즉시 저장. 엑셀처럼 빠른 데이터 입력이 가능합니다.

🖨

인쇄양식 빌더

국세청 표준양식 스타일의 인쇄 템플릿을 HTML로 작성. 자식 프로그램 데이터도 루프 출력.

🔌

서버 훅 시스템

16개 이벤트 훅으로 저장 전 검증, 자동 계산, 알림 발송 등 비즈니스 로직을 PHP로 추가.

📱

PC + 모바일 자동 대응

하나의 프로그램이 PC에서는 그리드, 모바일에서는 카드형으로 자동 변환. 토스/One UI 감성.

📊

간트차트 & 차트

프로젝트 관리용 간트차트, 드래그로 일정 변경. 다양한 시각화 도구 내장.

🔗

마스터-디테일 연동

부모-자식 프로그램 자동 연결. 탭으로 전환하며 관련 데이터를 한 화면에서 관리.

🌙

다크모드 & 테마

라이트/다크 모드 원클릭 전환. CSS 변수 기반으로 완벽한 테마 대응.

How it Works

3단계로 업무 시스템 완성

복잡한 설정 없이, 직관적인 화면에서 모든 것을 완성합니다.

① 메뉴 등록

프로그램명, 테이블명, 권한 코드만 입력하면 메뉴가 생성됩니다.

② 필드 정의

DB 필드명, 표시 이름, 타입, 너비만 지정하면 그리드+폼이 자동 구성됩니다.

③ 즉시 사용

별도 배포 없이 즉시 사용 가능. 수정도 실시간 반영. 개발-배포 사이클이 없습니다.

  • ✓ 코딩 없이 CRUD 자동 생성
  • ✓ 필터, 정렬, 페이징 자동 지원
  • ✓ JOIN 테이블 자동 연결 (FK 드롭다운)
  • ✓ 파일 첨부, 이미지 썸네일 자동
  • ✓ 서버 훅으로 비즈니스 로직 확장
  • ✓ 인쇄양식 템플릿 (국세청 스타일)
  • ✓ 실시간 목록편집 (엑셀 감각)
  • ✓ 간편추가 (대량 INSERT)
  • ✓ 모바일 자동 대응 (토스/One UI)
  • ✓ 다크모드 완벽 지원
